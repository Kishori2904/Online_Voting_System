# Online_Voting_System
A secure and efficient web-based voting platform built using PHP, MySQL, HTML, CSS, and JavaScript. It enables users to register, log in, and cast their votes online, ensuring transparency and one-person-one-vote authentication. Admins can manage candidates, monitor voting activities, and view live results through a dedicated dashboard.
